# 沖繩探索指南 2026（新北銀河 U10 移訓 · 個人空檔版）

一頁式、手機友善、可離線開啟的沖繩自由探索指南，涵蓋 7/3–7/8 各行程點周邊
**可步行 / 浮潛 / 街拍 / 參觀 / 在地美食** 的景點，由 AI 上網查證後彙整。

- 📄 主頁：[`index.html`](index.html)（單檔，直接用瀏覽器開）
- 🌐 線上版：GitHub Pages（見 repo 設定的 Pages 網址）
- 🔎 每個點附 Google Maps 連結、交通方式、最佳時間、費用、注意事項與查證來源
- ⚠️ 已設定 `noindex`，不被搜尋引擎索引（僅個人參考）

## 關鍵提醒

大本營飯店（ホテルみゆきビーチ）實際在**恩納村西岸**，比賽場地在**金武町東岸**，
兩地車程約 30–40 分。想「隨時跳海看珊瑚」的黃金點是**飯店自家海灘**；
金武比賽場地那側最近的 KIN サンライズビーチ**官方禁止浮潛**（只能戲水）。

## 如何調整內容

景點資料以 JSON 內嵌在 `index.html` 底部的 `<script id="data">` 區塊，
直接編輯該 JSON、存檔、`git push` 即可更新線上版。
